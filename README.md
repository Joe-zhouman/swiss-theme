# Swiss 主题说明

这是一款为 Typora 设计的个人化 **Swiss / International Typographic Style（国际主义平面风格）** 主题。它并不追求装饰性的视觉奇观，而是希望通过清晰的层级、克制的色彩、稳定的网格感与高可读性的排版，让内容本身更有秩序地呈现出来。

## 一、什么是瑞士风格

瑞士风格通常被称为 **国际主义平面设计风格**，形成于二十世纪中期的瑞士，并逐渐影响了全球的现代平面设计、出版设计、导视系统、企业视觉以及数字界面设计。

它的核心特征通常包括：

- 强调秩序、理性与信息清晰度
- 偏好无衬线字体
- 使用严格的对齐与网格系统
- 重视留白和结构，而非装饰
- 通过大小、粗细、位置建立层级
- 用少量高对比色作为节制的强调

这种风格之所以直到今天依然有效，是因为它非常适合承载信息密度较高的内容，例如文档、简历、研究笔记、项目说明与技术写作。

## 二、为什么把它用在 Typora 主题里

Markdown 写作天然偏向结构化表达，而 Typora 的优势在于“边写边看”。瑞士风格和这种使用场景非常契合，因为两者都关注一件事：**让内容的逻辑比装饰更先被看见**。

因此，这个主题的目标并不是做成“花哨的阅读皮肤”，而是做成一种稳定、耐读、适合长时间工作的写作界面。它既能服务技术文档，也能承载简历、作品集、研究报告、课程作业与项目展示。

## 三、主题的色彩系统

这个主题采用了相对克制的暖白底色与深棕红强调色组合。

### 背景色

整体页面背景不是纯冷白，而是略带纸张感的浅暖色。这样做有两个原因：

1. 长时间阅读时更柔和，不会像高亮冷白那样刺激
2. 能让白色内容卡片自然浮现，形成轻微但明确的层次

### 强调色

主题使用 `#80372b` 作为主要强调色。它不是鲜艳的纯红，而是更偏理性、沉稳、带一点印刷感的深红棕色。

这类颜色适合承担以下角色：

- 标题后的延伸线
- 大标题或列表的左侧强调线
- 普通白色卡片的外边框
- 局部重点标识

它的作用不是制造情绪，而是帮助建立结构。

### 内容底色

正文内容块和代码块以白色为主。白底让内容像被精确放置在版面中的信息单元，而不是漂浮的装饰块。白卡片与暖背景之间的轻微反差，是这套主题秩序感的重要来源。

## 四、主题的版式思路

### 1. 标题不是装饰，而是结构入口

这个主题中的标题承担非常明确的结构职责。

- `h1` 用更强的体量与左侧粗线承担“封面级标题”角色
- 其余标题不再做成大块卡片，而是转为“标题文字 + 右侧延伸线”的处理
- 延伸线不是修饰，而是在视觉上建立章节边界

这种处理方式会让页面看起来更像一份经过整理的版面，而不是传统博客页面。

### 2. 正文卡片承担信息容器角色

普通正文段落被包裹在白底红框卡片中，并支持连续段落自动连接。这样做的好处是：

- 段落不会散落在背景上
- 多段内容可以被看作一个完整信息单元
- 视觉节奏更稳定，更适合长文阅读

对于课程报告、项目说明、创作陈述这类内容，这种结构尤其有效。

### 3. 列表比正文更强调“条目感”

列表保留了左侧粗红线，因为列表天生具有“要点集合”属性，适合被看成更强的结构块。无论是实验目的、项目要点、技能标签还是结论摘要，列表都应该比普通正文更具轮廓感。

### 4. 代码区域延续克制原则

代码块采用白底、细线、直角、紧凑的行距处理，不使用夸张阴影和重装饰。这样可以保持技术内容与正文之间的统一秩序，同时确保代码依然易读。

## 五、这套主题适合什么内容

这份主题特别适合以下场景：

- 技术文档
- 课程作业
- 实验报告
- 研究笔记
- 简历与项目介绍
- 产品说明
- 作品集型 Markdown 页面

如果你写的是情绪化散文、文学性随笔、复古印刷风内容，这套主题未必是最合适的；但如果你的目标是清楚、稳定、可信、专业地组织信息，它会非常合拍。

## 六、这份主题的个性化取向

虽然它借鉴的是瑞士风格，但它并不是对历史样式的机械复刻，而是一种更适合当代屏幕阅读的个人化改写。

它保留了瑞士风格最重要的部分：

- 克制
- 网格感
- 清晰层级
- 功能优先
- 无衬线排版

同时也做了一些适应 Typora 使用习惯的调整：

- 让正文卡片更适合 Markdown 连续写作
- 让列表和标题承担更明确的结构角色
- 让代码区与正文区在气质上统一
- 用较柔和的暖底色平衡长时间阅读体验

## 七、如果你想继续微调

这套主题仍然可以根据个人习惯继续微调，例如：

- 调整标题延伸线粗细
- 收紧或放松卡片内边距
- 更换强调色为更冷或更亮的红色
- 调整列表项目符号的样式
- 增强表格与代码块的边界感

也正因为它的设计语言本身比较克制，所以小幅修改往往就能带来明显风格变化。

## 八、结语

这份 Swiss 主题的目标，不是让界面先声夺人，而是让内容显得更加可靠、有秩序、可持续阅读。它希望在 Typora 中提供一种接近现代平面设计版面的写作体验：安静、明确、克制，但不单调。

如果说 Markdown 是一种高效的写作语言，那么这套主题想做的，就是给这种语言配上一套同样清晰的视觉语法。
